# AI DevOps Failure Analyzer using Kiro CLI

## Overview
This project demonstrates an AI-powered DevOps troubleshooting solution developed using Kiro CLI.

The system analyzes DevOps failure logs and identifies:
- GitHub Actions failures
- Docker build failures
- Terraform failures
- Kubernetes issues
- NPM dependency conflicts

## Features
- Automated log analysis
- Root cause detection
- Suggested remediation
- Incident report generation
- Centralized DevOps troubleshooting

## Technologies Used
- Kiro CLI
- Python
- Git
- GitHub
- Terraform
- Docker
- Kubernetes
